Repositorio trabajado en clases de Desarrollo web + IA, que consta de una página HTML y CSS, a la cual se le aplicaron 
ciertas correcciones y mejoras en cuanto al apartado visual con CSS.

Principales correciones y mejoras:

- Se eliminaron etiquetas innecesarias de HTML como "hr" y "style", además de valores como "width" y "height".
- Creación de clases en CSS para optimizar código y estilizar la página.
- Agregar movimiento a las imagenes para hacerlo más interactivo.
